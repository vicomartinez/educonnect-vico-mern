# EduConnect

EduConnect es una plataforma educativa donde los estudiantes pueden compartir apuntes, resolver dudas y colaborar en grupo.  
Este proyecto se desarrollará con el **Stack MERN (MongoDB, Express, React, Node.js)**.

## Objetivo
Crear un espacio digital para fomentar el aprendizaje colaborativo entre estudiantes de preparatoria y universidad.

## Funcionalidades (MVP)
- Registro e inicio de sesión de usuarios.
- Publicación de apuntes o materiales de estudio.
- Comentarios en publicaciones.
- Perfil de usuario con datos básicos.
- Feed con publicaciones recientes.

## Tecnologías
- MongoDB (Base de datos)
- Express.js (Backend)
- React (Frontend)
- Node.js (Entorno)
- Git y GitHub (Control de versiones)

## Despliegue (planeado)
- Frontend: Vercel o Netlify
- Backend: Render o Railway
